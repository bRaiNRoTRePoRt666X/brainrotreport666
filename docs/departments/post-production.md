# Post-Production Department

## Role
Transform raw footage into polished, publishable episodes.

## Responsibilities

### Editing
- Assembly cut
- Fine cut
- Pacing optimization
- Continuity checks

### VFX
- Visual effects
- Motion graphics
- Compositing
- Green screen work

### Color
- Color correction
- Color grading
- Look consistency
- HDR/SDR conversion

### Audio
- Dialogue editing
- Audio mixing
- Noise reduction
- Loudness normalization

### Sound Design
- SFX selection
- Music integration
- Ambience
- Audio branding

### QC (Quality Control)
- Final review
- Technical checks
- Platform compliance
- Ethics verification

## Deliverables

| Item | Format | Location |
|------|--------|----------|
| Edited video | MP4 (ProRes) | `departments/post-production/editing/` |
| VFX elements | MOV (PNG) | `departments/post-production/vfx/` |
| Grade | LUT/MP4 | `departments/post-production/color/` |
| Audio mix | WAV | `departments/post-production/audio/` |
| Final export | MP4 (H.264) | `episodes/ready/` |

## Handoff Requirements

Before handing off to Distribution:
- [ ] Final export complete
- [ ] QC passed
- [ ] All versions rendered
- [ ] Metadata complete
- [ ] Legal compliance verified

## Tools
- Editing: Premiere, DaVinci, Final Cut
- VFX: After Effects, Fusion
- Audio: Pro Tools, Audition, Reaper

## KPIs
- Edit quality score
- Turnaround time
- Revision rounds
- QC pass rate (>95%)
