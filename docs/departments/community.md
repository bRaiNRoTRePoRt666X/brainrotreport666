# Community Department

## Role
Build and maintain relationships with the audience and partners.

## Responsibilities

### Engagement
- Comment responses
- Community posts
- Live stream interaction
- Fan recognition

### Moderation
- Comment moderation
- Spam removal
- Community guidelines enforcement
- Crisis management

### Feedback Collection
- Audience surveys
- Comment analysis
- Social listening
- Feature requests

### Collaborations
- Partner outreach
- Guest coordination
- Cross-promotion
- Influencer relations

## Deliverables

| Item | Frequency | Location |
|------|-----------|----------|
| Engagement report | Weekly | `departments/community/engagement/` |
| Moderation log | Daily | `departments/community/moderation/` |
| Feedback summary | Weekly | `departments/community/feedback/` |
| Collaboration tracker | Ongoing | `departments/community/collaborations/` |

## Handoff Requirements

From Distribution:
- [ ] Publish notification received
- [ ] Engagement plan ready
- [ ] Moderation guidelines set

To Creative:
- [ ] Audience feedback shared
- [ ] Content requests documented
- [ ] Collaboration opportunities identified

## Tools
- Management: Discord, Telegram
- Moderation: Native platform tools
- Surveys: Typeform, Google Forms

## KPIs
- Response rate (>50%)
- Response time (<2h)
- Sentiment score (>80% positive)
- Community growth rate
