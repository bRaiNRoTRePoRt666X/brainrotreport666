# Creative Department

## Role
Content conception, scripting, and visual design for all episodes.

## Responsibilities

### Scripts Team
- Episode outlines and full scripts
- Hook development (first 5 seconds)
- Call-to-action writing
- Fact-checking and source documentation

### Storyboards Team
- Visual planning for each scene
- Shot lists and camera directions
- Timing breakdowns
- Transition planning

### Graphics Team
- Lower thirds design
- Overlay creation
- Brand-consistent visuals
- Template maintenance

### Thumbnails Team
- Thumbnail design per episode
- A/B testing variants
- Brand consistency
- Click-through optimization

## Deliverables

| Item | Format | Location |
|------|--------|----------|
| Script | Markdown/PDF | `departments/creative/scripts/` |
| Storyboard | PDF/Images | `departments/creative/storyboards/` |
| Graphics | PNG/PSD | `departments/creative/graphics/` |
| Thumbnails | PNG (1280x720) | `departments/creative/thumbnails/` |

## Handoff Requirements

Before handing off to Production:
- [ ] Script approved by legal-compliance
- [ ] Storyboard reviewed
- [ ] All graphics ready
- [ ] Thumbnail drafts complete
- [ ] Ethics checklist signed

## Tools
- Writing: Google Docs, Notion
- Design: Photoshop, GIMP, Figma
- Storyboarding: Storyboarder, Photoshop

## KPIs
- Scripts delivered on time
- Revision rounds (target: <2)
- Thumbnail CTR (>5%)
