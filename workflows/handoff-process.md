# Department Handoff Process

## Overview

Episodes move through departments via structured handoffs. Each handoff
requires completion verification and ethics compliance sign-off.

## Handoff Stages

```
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
| INCOMING |────>|   CRE-   |────>|   PRO-   |────>|   POST   |────>|  READY   |
| REQUEST  |     |  ATIVE   |     |DUCTION   |     |PRODUCTION|     |          |
└──────────┘     └──────────┘     └──────────┘     └──────────┘     └──────────┘
                      │                │                │                │
                      ▼                ▼                ▼                ▼
                 Legal Review    Rights Check     QC Approval      Distribution
                 Ethics Sign-off  Safety Check    Final Export     Scheduling
